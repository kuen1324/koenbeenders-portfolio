# HERO_UPGRADE.md AI Workflow for Upgrading the Portfolio Hero to Award-Level

This document instructs AI agents how to systematically upgrade the portfolio hero section using the criteria defined in: Award-Level Portfolio Hero — Complete Design Criteria

The goal is to transform the hero into a **memorable, technically impressive, and visually disciplined portfolio introduction.**

---

# Objective
Upgrade the hero to meet **award-level portfolio standards**. This includes improvements in:
- concept
- layout
- typography
- motion
- interaction
- technical implementation
- performance
- accessibility

The process must be **iterative and evaluative**.

---

# Step 1 — Analyze Current Hero
Before changing anything, perform a full audit. Evaluate the hero using the criteria categories:

### Identity
- Is the creator clearly identified?
- Is the discipline obvious?

### Concept
- Is there one dominant idea?
- Or is the hero visually noisy?

### Hierarchy
- Is the scan order clear?
- Does the eye know where to go first?

### Layout
- Does the hero follow compositional logic?
- Is whitespace used effectively?

### Typography
- Is the type expressive?
- Is the hierarchy clear?

### Motion
- Is motion purposeful or decorative?

### Interaction
- Is interaction intuitive?

### Performance
- Does the hero load quickly?

### Accessibility
- Is the hero readable and accessible?

Produce an **audit report**. 
Format:
Hero Audit
Strengths: ...
Weaknesses: ...
Opportunities: ...

---

# Step 2 — Define the Hero Concept
Select **one dominant hero concept**. Allowed concepts:
- giant typography
- interactive 3D object
- cinematic visual stage
- kinetic typography
- interactive canvas
- editorial minimal layout

Rule: Hero = 1 dominant idea + 1 supporting system
Example: Dominant idea interactive WebGL object Supporting system minimal typography

---

# Step 3 — Define Hero Narrative
The hero must follow this story:
identity ↓ discipline ↓ visual concept ↓ interaction cue ↓ scroll into work

Example structure:
[Name / Brand]
Creative Developer
Building immersive digital experiences.
↓
Scroll

---

# Step 4 — Redesign Layout
Choose one layout type:

### Giant Typography
Name dominates screen.

### Split Layout
Text + visual object.

### Fullscreen Visual
Visual stage with overlay typography.

### Interactive Canvas
Entire hero is interactive.

Ensure:
- grid alignment
- clear hierarchy
- strong whitespace

---

# Step 5 — Upgrade Typography
Improve typography quality. Focus on:
- expressive typeface
- hierarchy
- spacing
- optical balance

Typography should communicate personality.

---

# Step 6 — Add Meaningful Visual
The visual must serve a purpose. Allowed roles:
- demonstrate technical skill
- create depth
- reinforce brand identity
- act as metaphor

Examples:
- 3D object
- generative particles
- shader material
- interactive element

Avoid purely decorative visuals.

---

# Step 7 — Implement Motion System
Motion must follow hierarchy:
1 intro presence 
2 headline reveal 
3 visual stabilization 
4 idle motion 
5 interaction feedback

Examples:
- split text reveal
- fade + translate
- slow floating motion
- parallax
- camera drift

Motion should guide attention.

---

# Step 8 — Add Interaction
Interaction should be discoverable.
Examples:
- hover feedback
- cursor magnetism
- object movement
- scroll transitions

Avoid interaction that blocks navigation.

---

# Step 9 — Optimize Performance
Hero must load quickly. Optimize:
- images
- video
- WebGL assets
- fonts

Avoid:
- layout shift
- heavy initial bundles

---

# Step 10 — Ensure Accessibility
Verify:
- readable contrast
- keyboard navigation
- reduced motion support
- click target sizes

---

# Step 11 — Static Frame Test
Take a screenshot of the hero. 
Ask: Does this still look strong without motion? If not, refine the layout.

---

# Step 12 — Final Evaluation
Score the hero. Rate each category from 1–10.

### Strategy
Identity clarity

### Concept
Originality

### Layout
Composition quality

### Typography
Craft and hierarchy

### Motion
Intentional animation

### Interaction
Discoverability

### Performance
Speed

### Accessibility
Compliance

If the score is below **8/10 in any category**, iterate.

---

# Final Goal
The hero should feel:
- memorable
- refined
- technically impressive
- minimal but powerful

It must demonstrate the creator’s craft within the first seconds of visibility.
